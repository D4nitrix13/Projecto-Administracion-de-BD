## Relaciones (Rombos/Diamantes)

Cada relaciГіn se dibuja como un **rombo** entre dos entidades con cardinalidades en las lГӯneas.

### 1. ROL вҖ” USUARIO

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”җ       в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ       в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ ROL  в”Ӯв”Җв”Җ(1,1)в—Ү PERTENECE в—Ү(0,n)в”Җв”Җв”ӮUSUARIOв”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ       в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ       в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

- Un ROL tiene **muchos** usuarios
- Un USUARIO tiene **un** rol

---

### 2. SECCIГ“N вҖ” USUARIO

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ     в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”ӮSECCIГ“N в”Ӯв”Җ(1,1)в—ҮASIGNADO_Aв—Ү(0,n)в”Җв”ӮUSUARIOв”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ     в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

- Una SECCIГ“N tiene **muchos** usuarios
- Un USUARIO tiene **una** secciГіn (nullable para admin)
